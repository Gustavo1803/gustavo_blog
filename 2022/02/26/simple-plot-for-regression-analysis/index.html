Simple plot for regression analysis

2022-02-26

In this post we are going to explore the ways to do regression analysis and how to plot them. We are going to use the famous dataset cars that has R. For the analysis we need to use a library called lm. We are going to check if the distance can predict the distance with the following model: \[dist = \beta_0 + \beta_1speed+\epsilon\]

This model can be done with the following command

model = lm(dist~speed,data=cars)
summary(model)

The model shows that one additional \(km/hour\) in the velocity increase the distance by 3.93 \(km\). To check the relationship we can plot the graphs with the following command:

library(ggplot2)
ggplot(cars, aes(x=speed, y = dist)) + geom_point()

It looks that there is a linear relationship. We can plot also the linear model that we just did to check the presition:

library(ggplot2)
ggplot(cars, aes(x=speed, y = dist)) + geom_point() + geom_smooth(method='lm')
## `geom_smooth()` using formula 'y ~ x'

The line of the regression analysis show that there is a linear relationship that the model show in the data.
